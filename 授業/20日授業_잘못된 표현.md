# 📘 일본어 단어 및 표현 정리

## 📌 전쟁·피난 관련 어휘
- 避難（ひなん）: 피난  
- 逃げる（にげる）: 도망치다  
- 噛む（かむ）: 깨물다  
- 戦う（たたかう）: 싸우다  
- 軍（ぐん）: 군대  
- 徴兵制（ちょうへいせい）: 징병제  
- 住み込む（すみこむ）: 고용주의 집이나 일자리에서 침식을 하며 지내다  
- せっかち: 성급함, 조급함  
- 死体（したい）: 시체  
- 集団（しゅうだん）: 집단

- 工夫（くふう）: 궁리, 고안  
- 加える（くわえる）: 더하다, 가하다  
- まろやか: 부드럽다, 순하다

- 増やす（ふやす）: 늘리다

## 📝 올바른 표현으로 바꾸기

| 잘못된 표현 | 올바른 표현 |
|-------------|-------------|
| 10年ぶり初恋に会いました | １０年ぶりに初恋（はつこい）の人に会いました |
| 彼女は涙が多いです | 彼女はよく泣きます / 彼女は泣き虫（なきむし）です |
| そんなに見えますか？ | そう見えますか？ |
| このお酒は果物の味がでます | このお酒は果物の味がします |
| パソコンで動映像をみています | パソコンで動画を見ています |
| 大学生ですか？じゃないと社会人ですか？ | 大学生ですか？それとも社会人ですか？ |
| 銀行で現金を探しました | 銀行で現金を下ろしました / 銀行で現金を引き出しました |
| 駅に行き方が分かりなせん | 駅までの行き方が分かりません / 駅への行き方が分かりません |
| 彼はクラスで一番人気が多いです | 彼はクラスで一番人気があります |
| 昨日のサッカーの試合は無勝負でした | 昨日のサッカーの試合は引き分け（ひきわけ）でした |
