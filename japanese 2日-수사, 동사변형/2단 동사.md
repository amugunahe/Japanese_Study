| 사전형 | 食べる (たべる)     | 먹다      |   |
| 부정형 | 食べない (たべない)   | 먹지 않다   |   |
| ます형 | 食べます (たべます)   | 먹습니다    |   |
| 과거형 | 食べた (たべた)     | 먹었다     |   |
| て형  | 食べて (たべて)     | 먹고      |   |
| 가정형 | 食べれば (たべれば)   | 먹으면     |   |
| 가능형 | 食べられる (たべられる) | 먹을 수 있다 |   |
| 명령형 | 食べろ (たべろ)     | 먹어라     |   |
| 권유형 | 食べよう (たべよう)   | 먹자      |   |
